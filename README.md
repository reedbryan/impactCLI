# impactCLI

A simple Java CLI application using picocli. Made for Impact.com by Reed Bryan #HireMe

## Requirements
- Java 8 or higher
- Maven

## How to Run

1. Clone the repo

   git clone https://github.com/your-username/impactCLI.git
   cd impactCLI

2. Build the project:
   
   mvn package

3. Run the CLI:
   
   java -jar target/impactCLI-1.0-SNAPSHOT.jar
