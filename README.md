# impactCLI

A simple Java CLI application using picocli. Made for Impact.com by Reed Bryan #Hire me

## Requirements
- Java 8 or higher
- Maven

## How to Run

1. Build the project:
   
   mvn package

2. Run the CLI:
   
   java -jar target/impactCLI-1.0-SNAPSHOT.jar

This will print the company name to the console.
